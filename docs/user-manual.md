## 1. 시작하기

### 1.1 첫 실행

1. **애플리케이션 실행**
   - 바탕화면의 "반편성 AI" 아이콘을 더블클릭합니다.

2. **초기 설정**
   ```
   ┌─────────────────────────────────┐
   │  🔐 초기 설정                    │
   ├─────────────────────────────────┤
   │  마스터 비밀번호를 설정해주세요  │
   │                                 │
   │  비밀번호: [____________]       │
   │  확인:     [____________]       │
   │                                 │
   │  ⚠️ 이 비밀번호는 데이터 암호화에│
   │     사용됩니다. 잊어버리면       │
   │     데이터를 복구할 수 없습니다. │
   │                                 │
   │  [다음]                         │
   └─────────────────────────────────┘
   ```

3. **학교 정보 입력**
   - 학교명, 학년, 반 개수 등 기본 정보를 입력합니다.

### 1.2 로그인

```
┌─────────────────────────────────┐
│  🔐 로그인                       │
├─────────────────────────────────┤
│  비밀번호: [____________]       │
│                                 │
│  [로그인]                       │
└─────────────────────────────────┘
```

## 2. 학생 데이터 관리

### 2.1 학생 데이터 입력 방법

#### 방법 1: Excel/CSV 파일 업로드 (권장)

1. **파일 준비**
   - Excel 또는 CSV 파일을 준비합니다.
   - 필수 열: 이름, 성별
   - 선택 열: 성적, 특성, 메모

   **예시 Excel 파일:**
   ```
   | 이름   | 성별 | 성적 | 특성        | 메모           |
   |--------|------|------|-------------|----------------|
   | 홍길동 | 남   | 85   | 리더십      | 반장 경험 있음 |
   | 김영희 | 여   | 92   | 조용함      |                |
   | 이철수 | 남   | 78   | 활발함      |                |
   ```

2. **파일 업로드**
   - "학생 데이터 관리" 메뉴로 이동
   - "📁 파일 업로드" 버튼 클릭
   - 또는 파일을 화면에 드래그 앤 드롭

3. **데이터 확인**
   - 업로드된 데이터를 확인합니다.
   - 오류가 있으면 수정합니다.

#### 방법 2: 직접 입력

1. "➕ 학생 추가" 버튼 클릭
2. 학생 정보 입력
3. "저장" 클릭

### 2.2 학생 데이터 수정

1. 수정할 학생 행의 "수정" 버튼 클릭
2. 정보 수정
3. "저장" 클릭

### 2.3 학생 데이터 삭제

1. 삭제할 학생 행의 "삭제" 버튼 클릭
2. 확인 메시지에서 "확인" 클릭

### 2.4 제약 조건 설정

**분리 희망 학생**
- 같은 반에 배치하지 않을 학생 쌍을 지정합니다.
- 예: 자주 다투는 학생들

**친구 관계**
- 같은 반에 배치할 학생 쌍을 지정합니다.
- 예: 서로 도움을 주는 학생들

## 5. 백업 및 복구

### 5.1 자동 백업 설정

```
┌─────────────────────────────────┐
│  💾 백업 설정                    │
├─────────────────────────────────┤
│  ☑ 자동 백업 활성화              │
│  주기: [매일 ▼]                 │
│  시간: [09:00]                  │
│  ☑ 백업 파일 암호화              │
│  보관 기간: [30일 ▼]            │
│                                 │
│  [저장]                         │
└─────────────────────────────────┘
```

### 5.2 수동 백업

1. "설정" → "백업 및 복구" 메뉴
2. "📦 지금 백업하기" 버튼 클릭
3. 설명 입력 (선택사항)
4. "확인" 클릭

### 5.3 복구

1. "설정" → "백업 및 복구" 메뉴
2. 백업 목록에서 복구할 백업 선택
3. "복구" 버튼 클릭
4. 확인 메시지에서 "확인" 클릭
5. 애플리케이션 자동 재시작

## 7. 문제 해결

### 7.1 자주 묻는 질문 (FAQ)

**Q: 비밀번호를 잊어버렸어요.**
A: 안타깝게도 비밀번호를 복구할 수 없습니다. 암호화된 데이터는 접근할 수 없게 됩니다. 정기적인 백업을 권장합니다.

**Q: 반편성이 너무 오래 걸려요.**
A: 학생 수가 많거나 제약 조건이 복잡할 경우 시간이 걸릴 수 있습니다. 제약 조건을 단순화하거나 AI 분석 상세도를 낮춰보세요.

**Q: Excel 파일을 업로드할 수 없어요.**
A: 파일 형식이 .xlsx 또는 .csv인지 확인하세요. 파일이 다른 프로그램에서 열려있지 않은지도 확인하세요.

**Q: 백업 파일은 어디에 저장되나요?**
A: `C:\Users\[사용자명]\AppData\Local\aichangeclass\data\backups` 폴더에 저장됩니다.

### 7.2 오류 메시지

**"데이터베이스 연결 실패"**
- 해결: 애플리케이션을 재시작하세요.

**"백업 복구 실패"**
- 해결: 백업 파일이 손상되었을 수 있습니다. 다른 백업을 시도하세요.

**"AI 모델 로드 실패"**
- 해결: Ollama가 실행 중인지 확인하세요.
